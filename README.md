# Прогнозирование биржевой цены акций на основе ансамблевой модели LSTM + XGBoost

## Общее описание

Методика краткосрочного прогноза биржевых цен с помощью стэкинг-ансамбля **LSTM + XGBoost** с доверительными интервалами на основе квантильной регрессии.  
Данные загружаются с Московской биржи (MOEX) и обогащаются фундаментальными показателями через T-Bank Invest API.  
Поддерживает два режима: прогноз на 1–3 дня вперёд и бэктест на исторических данных для проверки качества модели.

## Этапы методологии

### 1. Сбор данных

- Исторические свечи OHLCV с MOEX через библиотеку `moexalgo` (основной источник) и прямой MOEX ISS REST API (fallback)
- Фундаментальные показатели (P/E, P/B, ROE, Beta, DivYield, MarketCap) через T-Bank Invest API
- Период данных: с 2014-01-01 по текущую дату

### 2. Формирование признаков (Feature Engineering)

- **Технические индикаторы:** SMA(10), EMA(20), RSI(14), MACD + Signal + Histogram, Bollinger Bands, ATR(14), Stochastic K/D, ADX(14), Momentum(10), PriceChange(1d, 5d)
- **Фундаментальные признаки:** рыночная капитализация, рентабельность собственного капитала, дивидендная доходность, мультипликаторы P/E и P/B, бета-коэффициент
- Итого: **39 признаков** на каждый торговый день

### 3. Нормализация

- `MinMaxScaler` по всем 39 признакам
- Отдельный скейлер для целевой переменной `Close` (для обратного преобразования прогнозов)
- Скользящее окно `LOOK_BACK=30` дней → тензор формы `(N, 30, 39)`

### 4. Оптимизация гиперпараметров (Optuna)

- Байесовская оптимизация для LSTM: `units`, `dropout`, `learning_rate`
- Байесовская оптимизация для XGBoost: `n_estimators`, `max_depth`, `learning_rate`, `subsample`, `colsample_bytree`
- Результаты сохраняются в JSON и переиспользуются при повторных запусках

### 5. Walk-Forward обучение ансамбля

- 3 сплита с расширяющимся окном (80/85/90% — граница обучения)
- На каждом сплите обучаются:
  - **LSTM** (3 слоя, Dropout, EarlyStopping)
  - **XGBoost Regressor** (вход — сплющенный тензор 30×39 = 1170 признаков)
  - **3 квантильные XGBoost-модели** (α = 0.05, 0.50, 0.95) для доверительных интервалов
  - **Meta-Learner (XGBoost)** — объединяет предсказания LSTM и XGBoost
- Метрики качества: RMSE, MAE, R²

### 6. Прогнозирование

- Авторегрессивный прогноз на 1, 2 и 3 дня вперёд
- На каждом шаге пересчитываются технические индикаторы для нового ряда
- Доверительный интервал: 5-й и 95-й перцентили квантильных моделей

### 7. Оценка качества (Бэктест)

- Обучение на данных до указанной даты
- Прогноз на 1–3 дня вперёд и сравнение с известными реальными ценами
- Метрики: абсолютная ошибка (RUB), ошибка в %, покрытие доверительного интервала (CI Coverage)

## Ключевые навыки

- **Временные ряды:** walk-forward валидация, скользящее окно, предотвращение data leakage
- **Глубокое обучение:** LSTM, Dropout, EarlyStopping, Adam
- **Градиентный бустинг:** XGBoost, квантильная регрессия
- **Ансамблирование:** стэкинг, Meta-Learner
- **Оптимизация:** Optuna (байесовский поиск), персистентность гиперпараметров
- **Feature engineering:** технические индикаторы, фундаментальный анализ
- **Интеграция API:** MOEX ISS, T-Bank Invest REST API
