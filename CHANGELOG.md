## [v13.1] — 2026-04-22

### Добавлено
- `config.py` — все гиперпараметры LSTM и XGBoost вынесены из `.env` в Python-конфиг; безопасно коммитить
- `.gitignore` — исключены `.env`, директория `outputs/`, артефакты Python и IDE
- `CHANGELOG.md` — этот файл
- `architecture.md` — описание архитектуры проекта, потока данных, переменных окружения и порядка запуска
- `improvements.md` — анализ плюсов/минусов текущего решения и рекомендации

### Изменено
- `stock_modelv13.py` — гиперпараметры теперь импортируются из `config.py` вместо чтения через `os.getenv`
- `.env` — оставлены только API-токены (секреты); параметры моделей перемещены в `config.py`
- `README.md` — создан по образцу проектной документации

### Причина
Разделение секретов (токены) и конфигурации (гиперпараметры): токены не должны попадать в VCS,
гиперпараметры — наоборот, полезно версионировать для воспроизводимости экспериментов.

## [v13.0] — начальная версия

### Добавлено
- Стэкинг-ансамбль LSTM + XGBoost + Meta-Learner
- Квантильная регрессия для доверительных интервалов (α = 0.05, 0.50, 0.95)
- Walk-forward валидация (3 сплита с расширяющимся окном)
- Оптимизация гиперпараметров через Optuna с персистентностью результатов
- Режим бэктеста: обучение до заданной даты, сравнение с реальными ценами
- Загрузка данных MOEX через `moexalgo` + fallback на прямой ISS REST API
- Обогащение фундаментальными данными через T-Bank Invest API
- 39 признаков: OHLCV + 17 технических индикаторов + 17 фундаментальных
- Интерактивный режим ввода параметров и CLI-режим через `argparse`
